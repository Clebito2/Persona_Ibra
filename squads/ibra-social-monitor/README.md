# Ibrahim Social Monitor Squad

Squad especializado em monitoramento de redes sociais para a persona Ibrahim Boufleur.

## Agentes

1. **@ibra-screener**: Analista de conteúdo e sentimentos.
2. **@ibra-context-specialist**: Especialista em persona e posicionamento político/econômico.
3. **@ibra-responder**: Redator final (Estilo Ibrahim).

## Fluxo de Trabalho

O fluxo principal é o `social-response-workflow.yaml`, que coordena os três agentes para processar um link de postagem e sugerir respostas.
